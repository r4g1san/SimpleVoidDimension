# Simple Void Dimension

A simple Minecraft mod that adds a minimalist Void Dimension. Perfect for creative building or technical survival.

## Features
* **Void Dimension**: A peaceful, empty world with no land—only the void.
* **Dimension Block**: A special block used to travel between the Overworld and the Void Dimension.
* **Smart Teleportation**: Remembers your last position to ensure a safe return.
